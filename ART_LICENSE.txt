Pythoban Game Artist License

This document provides a list of image sources, design inspirations, and resources used in the development of the Pythoban game. We are grateful for the contributions of these artists and platforms that helped make this project possible.

Game and Platform Inspiration

1. Sokoban Online Game Platform
   - Website: [Sokoban Online](https://www.sokobanonline.com/)
   - Description: This platform provided the foundational concept and gameplay mechanics for the Sokoban puzzle design used in Pythoban.
   
2. Sokoban Origins and How to Play
   - Source: [Wikipedia - Sokoban](https://en.wikipedia.org/wiki/Sokoban)
   - Description: This page offered insights into the origins and rules of Sokoban, influencing the gameplay dynamics of Pythoban.

Development Tools and Documentation

3. Pygame Documentation
   - Website: [Pygame](https://www.pygame.org/docs/)
   - Description: This documentation was crucial for understanding the functions and classes used in the game development process.

4. Pytest Documentation
   - Website: [Pytest](https://docs.pytest.org/)
   - Description: Used for setting up testing frameworks to ensure game stability and performance.

Design and Visual Inspiration

5. Welcome Screen Design Inspiration
   - Source 1: [Adobe Stock - 8-bit Templates](https://stock.adobe.com/th/search/templates?k=8bit&asset_id=630846310)
   - Source 2: [Pinterest - Pixel Art](https://www.pinterest.de/pin/542050505152122609/)
   - Description: These sources inspired the retro 8-bit style welcome screen design for the game.
